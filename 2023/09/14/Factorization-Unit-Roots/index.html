因式分解与 n 次单位根

Posted on 2023-09-14 Edited on 2026-06-29 In Diary Waline:
高代小知识

习题引入

今天上高代的时候遇见这样一道题

用余数定理证明:(x4+x3+x2+x+1)∣(x5+x11+x17+x23+x29)\text {用余数定理证明:} (x^4+x^3+x^2+x+1)|(x^5+x^{11}+x^{17}+x^{23}+x^{29}) 用余数定理证明:(x4+x3+x2+x+1)∣(x5+x11+x17+x23+x29)

老师最后给的解答是这样的

对 (x4+x3+x2+x+1)(x^4+x^3+x^2+x+1)(x4+x3+x2+x+1) 进行因式分解，得到 (x−ω1)(x−ω2)(x−ω3)(x−ω4)(x-\omega_{1})(x-\omega_{2})(x-\omega_{3})(x-\omega_{4})(x−ω1​)(x−ω2​)(x−ω3​)(x−ω4​)

然后检验他们是不是右侧等式的根

当然这个解法没有问题，但是相信很多同学都有这样的疑问

  • 该怎么因式分解？
  • 一定要带进右边等式算吗？

本篇就是 epi 对这些问题的一点记录

单位根

首先，我们都知道 xn−1=(x−1)(1+x+x2+⋯+xn−1)x^n - 1=(x-1)(1 + x + x^2 + \dots + x^{n-1})xn−1=(x−1)(1+x+x2+⋯+xn−1)，注意到等式左边就出现了类似的结构。、

你一定知道 nnn 次多项式至多有 nnn 个根

有了这点共识，我们引入单位根

n 次单位根是 n 次幂为 1 的复数，它们位于复平面的单位圆上，构成正 n 边形的顶点，其中一个顶点是 1

比如，四次单位根有四个，是 1,i,−i，−11,i,-i，-11,i,−i，−1

有了单位根的概念，我们回头来看这个问题

问题解答

1+x+x2+x3+x41+x+x^2+x^3+x^41+x+x2+x3+x4 就对应了除 1 以外的五次单位根，也就是说，它们是单位圆的五等分点

而 x5+x11+x17+x23+x29=x4(x1+x6+x12+x18+x24)x^5+x^{11}+x^{17}+x^{23}+x^{29}=x^4(x^1+x^6+x^{12}+x^{18}+x^{24})x5+x11+x17+x23+x29=x4(x1+x6+x12+x18+x24)

我们可以类比的得到 x30−1=(x6−1)(x1+x6+x12+x18+x24)x^{30}-1=(x^6-1)(x^1+x^6+x^{12}+x^{18}+x^{24})x30−1=(x6−1)(x1+x6+x12+x18+x24)

所以等式的右端就对应着单位圆的三十等分点去掉六等分点，加上原点

根据几何直观可得，左边的根包含于右边，得证。

实际上，n 次单位根可以记为： 1,ε,ε2,ε3,…,εn1,\varepsilon,\varepsilon^2,\varepsilon^3,\dots,\varepsilon^n1,ε,ε2,ε3,…,εn （其中 ε=cos⁡2πn+isin⁡2πn\displaystyle{\varepsilon = \cos \frac{2\pi}{n}+i\sin{\frac{2\pi}{n}}}ε=cosn2π​+isinn2π​） 知道欧拉公式吧，还真有关系

于是我们可以做出这样的解答：

左侧的式子的根为 ε1,ε2,ε3,ε4\varepsilon^{1},\varepsilon^{2},\varepsilon^{3},\varepsilon^{4}ε1,ε2,ε3,ε4 （ε=cos⁡2π5+isin⁡2π5\displaystyle{\varepsilon = \cos \frac{2\pi}{5}+i\sin{\frac{2\pi}{5}}}ε=cos52π​+isin52π​），注意到 它们都是右侧式子的根，得证

看起来没有区别，但是实际上完全不用做计算就能搞定，感觉这种技巧也是不可不品的证明手段
