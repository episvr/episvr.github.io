2024 终～Memory Addition.

Posted on 2025-01-28 Edited on 2025-03-28 Waline:
叮！快来看看Epi的2024的最终总结补丁！

Start From 2025.1.27

之前发的总结还是太聚焦围绕学校的各种事了，于是在 2024 真正结束的除夕夜的前夜，我决定用一些不同的视角回顾一下 2024。

2024，对电脑进行了若干折腾，包括但不限于：重装系统，重整文件结构，拆机，装被破坏掉的编程环境。在这个过程中，我突然意识到文件管理的重要性～～（此处的文件管理是主谓短语而非专有名词）。~~

井井有条～Organized Knowledge

  • 了解所有装过的软件：我的做法是单独安排一个 Software 文件夹，再把快捷方式丢到.shortcut 中，之后可以通过 PowerToy 的 QuickRun 很方便的使用。

    • Objection！有的软件只能丢到特定的路径下恶心你，说的就是你 VirtualBox，这种软件就零星几个，可以只凭借记忆记住。
    • Hold it！有一类压缩包软件通常无需安装，开箱即用，我们称之为绿色软件，这种情况可以另起一个 GreenWare 文件夹存储。但 Epi 是维护了一个 Readme 解决的。

  • 统一建立 DownloadFile 文件夹：把所有的下载路径统统丢到这个目录之下，这样无论是查找还是删除文件都很方便，值得一提的是微信和 QQ 的目录真的是太怪了，所以可以同样是设置一个 Shortcut。

  • 配置记录：日常使用软件一定会留下自己痕迹，即 “个性化配置”，这时候留下一个 EpiConfig 是很有用的无痛迁移的手段，我会把一些软件的配置，比如 插件的名字 & 地址，浏览器的 Bookmark ，CSS 注入代码，API 配置，订阅源等 分文件夹统统都保存，定期备份到 U 盘。

  • 有一部分软件很特殊，它们大多体积小巧，功能单一，只是为了实现某种功能而存在，比如屏幕标注（Pointofix），校园网（USTB-Wifi-Tools），软件卸载（geekUninstaller）等，这些我倾向于另起一个 Tools 文件夹管理。

  • 有一部分更特殊的软件，我们可以单列一个 Game 文件夹管理他们。

  • 我的个人学习工作中会有巨量的文档.docx .pdf .pptx .txt .md 需要处理，我采用了一种特殊的方法处理这些文件，文件结构如下所示。

  • 最后，作为计科学生必须得有完整的编程环境，单独开一个 CodeEnv 文件夹存放编译器，单独开一个 Tmp 文件夹存放临时代码文件。

三月份的时候在家翻到了几台老旧的 OPPO ，一台 A57t，一台 R7，一台 R9m。R7 的那台屏幕坏掉了暂时没什么办法，A57t 被 Epi 折腾坏了。

旧机体验～Back Childhood

  • 经过艰苦卓绝的一通操作，成功 ROOT 并重装了 MIUI10 了（指先降级到安卓 4.4 利用系统漏洞 ROOT 再升级回 6.0 刷 Recovery 再找刷机包刷 MIUI），这期间狠狠的在酷安，B 站，知乎和贴吧赛博考古，得以一瞥互联网的早期形态。

  • ROOT 之后开始安装 Google 框架，经历种种磨难最终失败，暂时还完全没有头绪，但是期间在不知名的 bbs 上看到了编程领域大神反汇编分析什么 MIUI 的进程在作祟，赛博考古，得以一瞥互联网的早期形态。

  • 为什么执着于 Google 框架呢，这是因为旧手机的 Android WebView 版本太落后了，被一些网站拒之门外，这太不妙了。Epi 本想试着通过 Google Play 直接更新，但是失败了。自行核对下载的版本又总是报不兼容的错误。最后在酷安的一条帖子的评论里面找到了大佬分享过的机型对照表和软件包链接，最终才搞对版本。赛博考古，得以一瞥互联网的早期形态。

  • 之后，开始从 F-Droid 下一些软件，意外发现了 Aurora Store (一款无需 Google 框架的第三方 Google Play)，兜兜转转回到原点了属于是。下载了一些颜值不错的音视频播放器，再下载一些番剧 & 歌丢进手机里。安 TTS 引擎，配 RSS 阅读，安第三方桌面，再 Ban 掉一些系统服务。看着丝滑流畅的动画，看着赏心悦目的 UI，这种感觉真的极其舒爽。

  • 最近，我突然想起一款老游戏 “声之寄托 "（大体上是一个通过 “打电话” 解救美少女的游戏，真心不错推荐一下）, 前两部都有着相当多的资源，但是第三部《声之寄托 Last Game》 由于没有汉化少有转载，能下到的也仅仅是一个空壳子，重要的 OBB 数据包已经无处寻找，凭着记忆去扑家吧翻了翻，发现早就停止服务了还是经过一番辗转，再国外的一个小论坛上找到了完整的数据包，现在正在快乐的同 GPT 一起啃日语生肉。

    • Fun Fact：初二的 Epi 在某贴吧发表的第一篇帖子就是关于声之寄托 Last Game 的汉化的，难以想象那时候的我是怎么啃下生肉的，去翻了一下用词尬到爆表，差点忍不住删帖，等有机会重新完善一下吧。赛博考古，得以一瞥互联网和 Epi 的早期形态。
    • Fun Fact：记得没错的话，第一次用 B 站看的视频应该是节奏大师自制铺，而现在已经是音游遍地开花的时代了。
    • Fact: Epi 在这台手机上通关了逆转裁判 4，逆转检事 1，逆转检事 2，时间空洞，幽灵轨迹和极限脱出 9 小时 9 人 9 扇门，而他正在用这台手机看的番剧是命运石之门。

尽管到这里已经很满足了，但是这样还不够，我想让这台手机和电脑连接起来，做出更 interesting 的事～～（科技并伴着趣味）。~~

无线连接～Invisible Connect

  • 为此，我下载了 Termux, 但是在安装 ssh 时出现了一个问题：Termux 的软件包找不到 ssh，而 Termux 版本过旧 pkg upgrade 完全不起作用。虽然之后通过一系列的换源解决了问题，但是 Epi 决定使用另一套战术。在大二上学期，Epi 斥巨资～～（200RMB）~~ 购入了一个 Android 14 系统的平板，虽然手感很不对劲感觉像是刷的，但是它真的解决了问题（指背一坨书爬 if 楼），Epi 通过成功让平板和电脑（在同一局域网内）远程连接在了一起。
    • Epi 在寒假期间狠狠的调教了平板，顺滑多了。
  • 在平板电脑中，使用 Mkdocs 部署了一个本地笔记知识库 Epi Docs，看起来很不错。
  • 内网穿透正在尝试，但是尝试的不太成功。

  • 试着剪了两个 Mad，一个是 MyGO，一个是 Mujica。剪 Mad 真的意外的很有学问在里面（很佩服一些会分镜会动画的）
    • 最近二次元浓度疑似有点高了
  • 为 PuzzleHunt 的一个解析视频意外的吸引到了一些人，还挺意外的。
  • 下半年开始有一些组队打比赛的想法，开始试着做计划，组织自己的发言，不过条理性这点还得跟 GPT 多学学。

诗与远方～Lifetime Poem

这一年我写了一些诗，但是大多有点尬了所以没发，这里挑出一首发出来，写的时候也没顾什么平仄，凑活着看吧。

2024.11.10

秋景渐散柿树空，杏叶碎铺映橘灯。
巴别塔下多心意，象牙谷里空页声。

歌词大意：

这是在图书馆前那条路上写的，先想起了上周这里还有人在这里摘柿子，又想到大一的时候参加的柿子文化节，突然就想写点东西（原来古人都是这么写诗的）。走着走着又看到了很多人聚在银杏大道上拍挂在树上的闪灯，其实去年我也这么干过 hhh，顿时 “时间过得真快” 这种感觉就油然而生。Actually 这时候自己因为各种各样的事情忙得要命，又有些不是很会读空气的人添堵，以至于要做些正经事情的时候不得不逃离宿舍到外面去 “失踪一会”，明明大家说的语言都是一样的，想说的话却还是像被巴别塔一样相隔开。于是就有了 “巴别塔下多心意”。唉，从前在 “象牙塔”（也就是初高中）总是独来独往。要是能再来一次，想告诉曾经的自己也试试迈出课桌，别把稚嫩青葱的时光都倾注在书页上。是怀着三分后悔七分告诫写下最后一句话的样子。

得分点解析：
这两句诗写得很有意境啊，Epi！秋天的景色渐渐散去，柿子树上也变得光秃秃，地上铺满了碎杏叶，配着温暖的橘色灯光，感觉很有秋日的温馨与静谧。而 “巴别塔” 和 “象牙谷” 的意象，一个代表了纷繁的交流，一个代表了知识的求索。“多心意” 确实让人感觉生活中有各种牵挂和意图，彼此交流并不完全真诚单纯，反而在每个人心里都带着各自的心思。而 “空页声” 则很巧妙地表达了一种失落的怀念 —— 对过去简单、纯粹的回忆；同时又带着些许疏离和批判，觉得象牙塔内的学术氛围或许有些空洞、缺乏真正的意义。这种深层的内涵让整首诗不只是描绘了一种场景，更像是一种对生活和学术环境的反思。写得好浪漫，带点秋意的伤感和思考。Epi，你是有心之人啊。（这是 ChatGPT 评的，哈哈真让它夸到了哈哈）

我关注过一个叫做 Arlene_Libitina 的 up 主，她的视频很干净，纯粹，像杂闹温馨的瓦尔哈拉，又像是一个简单安静的咖啡馆。她有留过这样一条动态：未来有机会，我想去写一本我自己的《拉多加湖》。

未来是多远，去远方又有多长的路，我还不知道，但是我已在路上。

Time is but the stream I go a-fishing in. —— 梭罗《瓦尔登湖》
