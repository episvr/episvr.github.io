一次关于 Thinkpad T420 的升级

Posted on 2026-06-29 In Diary Waline:
对 ThinkPad T420 进行 CPU、内存、散热、C壳 的全面升级改造记录

前情提要：Install Debian on ThinkPad-T420

之前估计的运行强度好像低了一点，实际用这台笔记本工作的时候总觉得不太跟手，加上第一次清灰的时候 不小心 搞坏了导热泡棉，于是在长达四个月的思想斗争后，终于找到了契机（借口），下定决心做一次升级改造。

但是最主要的原因是撕开了 C 壳的一个无关紧要的小口，好难看受不了了。

购买的东西

  • 带指纹模块的 C 壳（原来这个也是模块化的！）
  • 一个更强力的三铜管散热
  • i7-2720qm cpu
  • 两块 8G 三星 DDR3L 1600 低压内存

现在是 2026.05.06 15:29，就让我期待一下所有的东西吧

现在是 2026.05.13 15:29，很不幸的是，我购买的 C 壳的触摸板坏掉了，尽可能找了商家重发。

【然后就忙的没时间写了，乐】

现在是 2026.06.29 21:13，我终于想起来还有这么一回事，我决定把这篇文章写完。

拆机

说简单也简单，说麻烦也麻烦。T420 的拆机流程是这样的：

  1. 把背板上的所有螺丝拆完
  2. 翻到正面，轻轻向下推一下键盘，拆开排线。
  3. 把正面现在能看到的三个固定螺丝拆了
  4. 小心地把 C 壳卸下来。
  5. 完成！

升级

内存的升级很简单，一块在背板，一块在 C 壳的下面。拆完安上就行。

CPU 的升级也还行，把散热模组的螺丝拧下来就可以看到被固定住的 CPU 了，之后拧开右侧的限位螺丝就行。

指纹模块被安装在 C 壳的背面，简单安装一下排线即可。

然后就是反着做一遍上述流程就行，皆大欢喜，皆大欢喜。

这…… 当然是不可能的。

教训

  1. 在升级 CPU 后，第一件事就是无法在没电池的情况下开机，这是因为官配的 65W 电源带不动 CPU，需要换成 90W 的电源。
  2. 升级 bios 会 不可逆地 炸掉 Thinkpad 触控板和小红点的驱动，需要重新配驱动，如果现有的情况稳定的话不要盲目升级，我才没有踩坑我猜的。
  3. 指纹模块超级垃圾，识别准确度很低，不要买。如果你不幸买了，可以调整指纹模块源代码里的相似阈值，降到 20 会好很多。
  4. 总之，我只建议升级内存，升级其他的东西提升不是很明显，只能说又花钱买教训了😢。
  5. 在我的使用场景，我用着差不多还行了。（我没意见）
