数据结构之线性表

线性表是由相同类型数据元素组成的有限序列
顾名思义，就像是一条线，除了端点，每个元素都有唯一的直接前驱，直接后继

线性表有两种存储方式

1 顺序存储（顺序表）

顺序存储中元素的存储是连续的，这样的优点就是可以很快的访问元素，但是删除和插入要移动大量元素
顺序存储可以分为以下两类

1.1 静态分配

使用数组data[]存储数据，最大空间为Maxsize,实际存储的元素个数为length

const int Maxsize = 100; // 设置最大空间为100

struct StaticArray 
{
    int data[Maxsize]; // 数组存储数据
    int length; // 实际存储的元素个数
};

容易超出分配的内存空间（溢出）

1.2 动态分配

使用一段连续的存储空间的首地址存储数据，大小为Maxsize,实际存储的元素个数为length

struct DynamicArray 
{
    int *data;   // 指向动态数组的指针
    int maxSize; // 顺序表的最大容量
    int length;  // 实际存储的元素个数
};

1.2.1 创建与初始化

  • 分配空间
  • 是否分配成功
bool init(DynamicArray &arr, int maxSize)

int locatedata(DynamicArray arr，int tobefind)
{
    for(int i = 0; i < arr.length; i++)
    {
        if (arr.data[i] == tobefind) return i+1;
    }
    return -1;
}
