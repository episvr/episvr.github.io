选择排序 && 冒泡排序

2023-12-01

写在前面的话

在写排序的时候，务必仔细检查数组的下标

选择排序

选择排序的重点在于 选择，即找到 最值元素
我们可以建立一个标记flag,让其记住最值元素的下标

  • 例：使用选择排序对长度为n的数组由大到小排序
int flag,min;
for(int i = 0; i < size - 1; i++)          //思考这里 i < size-1 的含义;
{
    flag = i;                        //思考这里 flag = i 的含义;
    if (int j = i+1; j < size; j++)  //思考这里 j = i+1 的含义;
    {
        if (x[j] > x[flag])
            flag = j;             //使用flag标记最大值的下标
        if(flag != i)
        {
            min = x[i];
            x[i] = x[flag];       //swap(x[i], x[flag]) 
            x[flag]=x[i]
        }
    }
}

冒泡排序

选择排序的重点在于 交换，即不断进行交换，让数组逐渐趋近于有序状态（最值会通过交换 "冒" 出来）

for (int i = 0; i < size - 1; i++) 
{
    bool IsSwap = false;                   //思考:为什么要用isswap？
    for (int j = 0; j < size - i - 1; j++) //思考:为什么是size-i-1？
    {
        if (a[j] > a[j + 1]) 
        {
            swap(a[j], a[j + 1]);
            IsSwap = true;
        }
    }
    if (!IsSwap) break;
}
